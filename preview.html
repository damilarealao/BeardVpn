Changes from current build

  • Home top bar: Removed the fake blue shield icon. Just clean "BeardVpn" text.
  • Logo: Using your actual Viking shield logo (assets/icon.png) - no background, centered.
  • Removed: The padlock emoji, the triangle arrow, the floating overlay hack.
  • Ad flow (rewarded): Tap CONNECT → full-screen ad pops up → 5s countdown → auto-close → VPN connects. No manual close.
  • Settings: Added Developer section with email + GitHub. Kill Switch & Auto-Connect are now stateful.
  • Crash fix: startForeground() called immediately in onStartCommand().
  • VPN permission: Uses startActivityForResult + ActivityEventListener properly.
  • ConnectButton: Spinning loader animation now actually rotates.
  • MonetagAd: Loading indicator + error handling in WebView.
  • HomeScreen: Shows "Loading servers..." when server list is empty during initial fetch.
  • ServerList: Fixed double padding (removed marginHorizontal from ServerCard).
  • useVpn: Stats interval no longer leaks on reconnect. DNS from settings is passed to native.
  • BeardVpnService: Detailed logging for every state change.
  • formatSpeed: Returns "—" instead of "0.0 Kbps" when not connected.
  • State sync fix: useVpn queries getStatus() on mount AND when app comes to foreground (AppState listener). VPNModule receiver registered in init{} not on connect.
  • Disconnect fix: Disconnect now immediately emits "disconnected" to JS (no more stuck states).
  • Server list fix: rankServers() no longer filters out servers with ping=0 or speed=0. All servers now shown, sorted by score.
  • Server loading fix: Cached servers from AsyncStorage shown instantly on launch. Fresh fetch happens in background. Bundled 10-server fallback if all fetching fails.
  • OpenVPN client: Still uses custom SSLSocket. Traffic is NOT tunneled yet — key icon shows because TUN interface is established, but routing doesn't work. Next step: integrate ics-openvpn or switch to L2TP/IPSec.
